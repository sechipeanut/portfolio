# React Icons Reference for Tech Stack

## üé® How to Add Tech Stack Items in Sanity

When adding a tech stack item in Sanity Studio:

1. **Name**: The technology name (e.g., "React", "TypeScript")
2. **Icon Library**: Choose from:
   - `si` - Simple Icons (most tech logos)
   - `fa` - Font Awesome (general icons)
   - `di` - Devicons (developer icons)
   - `tb` - Tabler Icons (modern, customizable icons)
3. **Icon Name**: The exact component name from react-icons
4. **Color**: Hex color code (e.g., `#61DAFB` for React blue)
5. **Order**: Display order (0, 1, 2, etc.)

## üí° Tips

1. **Icon Library Prefix**:
   - Simple Icons: All start with `Si` (e.g., `SiReact`)
   - Font Awesome: Start with `Fa` (e.g., `FaNodeJs`)
   - Devicons: Start with `Di` (e.g., `DiGit`)
   - Tabler Icons: Start with `Tb` (e.g., `TbBrandReact`)

2. **Naming Convention**:
   - Remove spaces and special characters
   - Use camelCase after the prefix
   - Example: "Next.js" ‚Üí `SiNextdotjs`

3. **Color Matching**:
   - Use official brand colors (hex codes)
   - Visit https://brandcolors.net/ for official brand colors
   - Icons will display in their native colors for a clean, professional look
